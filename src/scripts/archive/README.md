# Archived resumes (do NOT use)

`single-ComVis.tex` and `single-GenMod.tex` are old example outputs kept only for historical
reference. They predate the current conventions (named vspace variables, the canonical
Publications block, several profile corrections) and contain known errors.

NEVER seed a new resume from these, and do not read them when tailoring. Always start from
`src/single-ref.tex` (the canonical seed). See CLAUDE.md.
